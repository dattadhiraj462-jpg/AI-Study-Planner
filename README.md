# AI-Study-Planner
An intelligent and responsive study planner built using HTML, Tailwind CSS, and JavaScript that helps students generate personalized study schedules based on subject priorities and exam preparation needs.

1. Features
. Priority-based study allocation
. AI-style smart schedule generation
. Built-in Pomodoro-inspired sessions
. Performance dashboard & study analytics
. Mock test / exam mode
. Automated break scheduling
. Fully responsive modern UI
. Real-time study session tracker

2. Tech Stack
HTML5
Tailwind CSS
JavaScript

3. Purpose

Designed to help students maximize productivity and maintain balanced study sessions with smart time management.
